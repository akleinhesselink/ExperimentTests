#### Data logger metadata 9/12/2012 ####

Data loggers record the temperature and soil moisture at the USSES precipitation field experiment. CSV files in this directory correspond to the loggers at two blocks of plots.  

A.K. collected loggers on 9/11/2012 and downloaded the data from the loggers on 9/12/2012

Port 1: A -- temperature with ECT sensor
Port 2: B -- 5cm soil moisture west side of plot EC5 sensor
Port 3: C -- 25cm soil moisture west side of plot EC5 sensor
Port 4: D -- 5cm soil moisture east side of plot EC5 sensor
Port 5: E -- 25cm soil moisture east side of plot EC5 sensor

15_16_control.csv -- logger for the control plot for pair 15/16, Device ID: EM20067 
plot_15_soil_moisture.csv -- logger for plot 15, Device ID: EL5741
plot_16_soil_moisture.csv -- logger for plot 16, Device ID: EL5743

1_2_control.csv -- logger for the control plot for pair 1/2, Device ID: EL5739
plot_1_soil_moisture.csv -- logger for plot 1, Device ID: EL5742
plot_2_soil_moisture.csv -- logger for plot 2, Device ID: EL5740
